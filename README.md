Como subir la pagina
Primero debemos crear un repositorio en nuestra cuenta de GitHub, en mi caso llamado Gustavo_Lopez_2023101538, luego copiamos el enlace del repositorio para utilzarlo mas adelante. Luego vamos a git bash y ejecutamos creamos la carpeta donde se alamacenara los archivos en este caso con TEMA 1 y TEMA 2, desarrollamos el codigo la pagina web, en la carpeta tema 2, y luego desde la terminal git push, ejecutamos, git init, luego git add ., luego git commit -m con el mensaje, git remote add origin https/....git, y git push -u origin master con ello tenemos nuestros archivos en el repostitorio
